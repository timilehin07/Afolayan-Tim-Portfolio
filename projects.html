    Interactive Snake Cursor - 2025

    Interactive snake cursor demo built with HTML5 Canvas and vanilla JavaScript. Showcases smooth, physics-based animation, trigonometry, and modular OOP design. A creative frontend project demonstrating advanced cursor interaction and canvas rendering.

    Weather Forecast App - 2025

    A responsive weather application built with React.js that fetches real-time data from OpenWeatherMap API. The app detects user location via browser geolocation and displays current weather conditions including temperature, humidity, wind speed, and atmospheric pressure. Includes a 5-day forecast with visual weather representations and responsive design that works seamlessly across all devices.

    Tsuup - 2024

    The Tsuup application, developed as my final year project at Cavendish University Uganda, provides users with a comprehensive solution for accessing Egusi recipes and ordering the necessary ingredients. The app offers a user-friendly interface where users can easily browse different Egusi dishes, read recipes, and conveniently order the ingredients needed for preparation. It aims to simplify the cooking process and introduce the richness of Egusi cuisine to a wider audience.

    Smart Fan System - 2023

    This project was developed as my Third year project at Cavendish University Uganda. It is a web-based control system for a smart fan using HTML, CSS, and JavaScript. I implemented manual and voice control functionalities for turning the fan on/off and adjusting speed. The system is designed to improve user comfort by enabling remote control of fan settings via both manual and voice commands. This project demonstrates the integration of web technologies with smart devices, offering users more control over their environment.

    Online E-Ticket System - 2021

    The Online E-Ticket System, developed as my first-year project at Cavendish University Uganda, is a comprehensive web-based platform designed to book flight tickets. This innovative system provides travelers with a convenient and efficient way to manage their travel arrangements, from selecting flights to making secure online payments. Targeting both individual travelers and travel agencies, the system integrates advanced technology to enhance user experience and operational efficiency. It simplifies the process of booking, ticketing, and managing flight details online, making it more accessible and user-friendly.
